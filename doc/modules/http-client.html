HTTP 客户端

src/utils/http/index.ts — 全项目唯一 Axios 实例。

配置

  • baseURL ← import.meta.env.VITE_SERVER_BASEURL（去尾斜杠）
  • timeout：30000 ms
  • paramsSerializer：qs.stringify（兼容 pure-admin 分页参数）

鉴权白名单

以下路径不附加 Bearer（也跳过 refresh 逻辑）：

/admin/auth/login
/admin/auth/dev-login
/admin/auth/refresh

请求拦截

  1. 读 Cookie/localStorage getToken()
  2. 若 access 未过期 → 设置 Authorization: Bearer <accessToken>
  3. 若过期 → 单飞 handRefreshToken，队列等待新 token 后重放请求

响应拦截

  • 成功：返回 response.data（非完整 AxiosResponse）
  • success === false → message(msg) + reject
  • HTTP 401/403 → removeToken()
  • 错误体 msg 字段 → toast 提示

对外 API

import { http } from "@/utils/http";

http.get<T>(url, { params })
http.post<T>(url, { data })
http.request<T>(method, url, config)

业务模块应通过 src/api/* 封装，而非页面直接拼 URL。
