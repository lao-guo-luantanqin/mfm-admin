集合文档 CRUD

白名单 Mongo 集合的通用表格 + JSON 编辑器，替代原 uniCloud clientDB 运营操作。

页面

src/views/business/collections/index.vue（路由名 BusinessCollections）

  • 集合下拉：来自 GET /admin/collections
  • 表格列：首行样本动态推断 + 固定 _id
  • 搜索：keyword 查询参数
  • 新建/编辑：对话框内 JSON 文本，提交前 JSON.parse 校验
  • 删除：ElMessageBox.confirm 后 DELETE

API 封装

src/api/admin/collections.ts

函数 HTTP
fetchCollectionMeta GET /admin/collections
fetchRecordsPage GET /admin/collections/{name}/records
saveRecord POST 或 PUT（有 docId 时）
deleteRecord DELETE

后端

  • 路由：mfm_api/api/admin/collections_router.py
  • 服务：mfm_api/services/admin/collections/crud.py
  • 契约：mfm_api/schemas/admin/collections.py
  • 白名单：与 sync-batch 相同 env（MFM_API_SYNC_BATCH_ALLOWED_COLLECTIONS）

局限与后续

  • 当前为通用 JSON 编辑，无字段级表单校验与关联选择器
  • 复杂集合（如嵌套图片、关联 ID）运营体验有限，可按集合拆专用页
  • 写操作校验以后端 Beanie / sync-batch 规则为准

验收清单

  1. 后端 env 含目标集合名
  2. 登录后菜单可见「集合文档」
  3. 列表分页、keyword 搜索正常
  4. 新建/编辑/删除后列表刷新
  5. 非白名单集合不可写（403/404）
